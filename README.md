# Sai's Archive
A personal website for sharing who I am, what I’m building, what I’m learning, and the ideas I’m currently exploring.

## Overview
Sai’s Archive is my personal website and portfolio. It brings together the projects I’m building, things I’m learning, blog-style notes, and interests I want to document properly.

## Why I Built This
I wanted a central hub for presenting my projects and interests in a way that feels more personal than a github profile or a resume alone.

## Current Features
- Header navigation
- About section
- Project preview section
- Blog preview section
- Contact/footer section
- Responsive layout in progress
